# Sales-Data-Analysis-SQL-Python-Excel
## Sales Data Analysis project using SQL, Excel, and Python to clean, analyze, and visualize sales data and generate business insights.

## Project Overview
This project performs end-to-end Exploratory Data Analysis (EDA) on a sales dataset using Python libraries such as NumPy, Pandas, Matplotlib, and Seaborn.
The objective is to extract meaningful business insights, identify trends, and visualize data for decision-making.
